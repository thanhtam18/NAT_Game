## 2. AXP202 Power domain

|   CHANNEL    |    Explanation     |
| :----------: | :----------------: |
| AXP202  DC2  |       NO USE       |
| AXP202  DC3  | ESP32(Can't close) |
| AXP202  LDO1 |   Can't control    |
| AXP202  LDO2 |     Backlight      |
| AXP202  LDO3 |     TFT/TOUCHT     |
| AXP202  LDO4 |     GPS Module     |
